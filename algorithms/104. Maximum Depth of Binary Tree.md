## 104. Maximum Depth of Binary Tree

### 题目分析
求一颗二叉树最大的深度。

### 解法
递归去求左右的深度，然后用大的那个+1。

时间复杂度O(N)

空间复杂度O(1)

### 代码
```



# Definition for a binary tree node.
# class TreeNode:
#     def __init__(self, x):
#         self.val = x
#         self.left = None
#         self.right = None

class Solution:
    def maxDepth(self, root):
        """
        :type root: TreeNode
        :rtype: int
        """
        if not root:
            return 0
        return 1 + max(self.maxDepth(root.left),self.maxDepth(root.right))

```
