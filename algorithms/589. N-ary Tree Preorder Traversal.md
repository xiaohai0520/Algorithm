## 589. N-ary Tree Preorder Traversal.py


### 题目分析

先序遍历一颗N-ary树


### 解法
略


### 代码

递归
```
"""
# Definition for a Node.
class Node:
    def __init__(self, val=None, children=None):
        self.val = val
        self.children = children
"""

class Solution:
    def preorder(self, root: 'Node') -> List[int]:
                
        if not root:
            return []
        res = []
        def helper(node):
            if not node:
                return
            res.append(node.val)
            for n in node.children:
                helper(n)
        helper(root)
        return res
```

非递归
```
"""
# Definition for a Node.
class Node:
    def __init__(self, val, children):
        self.val = val
        self.children = children
"""
class Solution:
    def preorder(self, root: 'Node') -> List[int]:
        
        if not root:
            return []
        stack = [root]
        res = []
        while stack:
            cur = stack.pop()
            res.append(cur.val)
            
            if cur.children:
                stack.extend(cur.children[::-1])
        return res
```
