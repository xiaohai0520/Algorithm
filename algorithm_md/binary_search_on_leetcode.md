# Leetcode Binary Search problems 

## Sorted Matrix
二分查找经常会用在一维的数组中，当一个二维的矩阵的行和列也是单调排列的时候，我们这个时候也是可以考虑二分法的。
这类题的解法往往都是从矩阵的左下角或者右上角开始搜索，每次进行行更新或者列更新，直到矩阵的另外一个对角结束。
